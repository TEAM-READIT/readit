# 1. 개요

## 프로젝트 소개

- Readit은 뉴스,비문학 읽기 및 요약을 통해 문해력 향상에 도움을 주는 서비스입니다.
- 챌린지를 통해 본인의 실력을 확인하고, 커뮤니티를 통해 함께 문해력을 향상할 수있습니다.
